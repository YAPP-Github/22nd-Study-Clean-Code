## 📌 진행 방식
- 1주일 동안 한 장씩 모두가 읽기
- 한 주에 한 명씩 번갈아가며 정리한 내용 공유
- 정리한 내용은 `.md` 파일로 각 장 폴더에 업로드
- 매주 토요일 11시에 모여서 1시간 진행(일정에 따라 유동적으로 진행 예정)
- 모두가 **부담없이** 스터디에 참여하자
